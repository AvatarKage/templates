`npm run init` to install dependencies and complete setup<br>
`npm run sandbox` to test code in `/sandbox`<br><br>

`npm run start` to run from `/dist`<br>
`npm run dev` to develop `/src`<br>
`npm run backup` to backup code to `/dev/backups`<br>
`npm run sync` to sync `/config/config.toml` to `package.json` and translate `/config/lang`<br>
`npm run build` to compile to `/dist`<br>
`npm run clean` to delete `/dist` and `/logs`
